## 📖 Overview

This project implements a Coverless Image Steganography technique using a combination of SIFT (Scale-Invariant Feature Transform) and ORB (Oriented FAST and Rotated BRIEF) feature extraction methods.

Unlike traditional steganography, the proposed approach focuses on secure information hiding while preserving image quality and improving resistance against steganalysis attacks.
